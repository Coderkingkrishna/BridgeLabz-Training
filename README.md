## DBMS-CSharp-Practice

### Day 1: SQL Fundamentals - Core Database Languages

#### 🎯 Topics Covered
- DDL (Data Definition Language) - CREATE, ALTER, DROP, TRUNCATE
- DML (Data Manipulation Language) - INSERT, SELECT, UPDATE, DELETE, JOIN, MERGE
- DCL (Data Control Language) - CREATE LOGIN/USER, GRANT, DENY, REVOKE
- TCL (Transaction Control Language) - BEGIN, COMMIT, ROLLBACK, SAVEPOINT

#### 🧪 Problems Implemented
- Database and table creation with constraints
- CRUD operations and data retrieval
- Complex queries with JOINs
- User authentication and permission management
- Transaction management with save points
- MERGE operations for data synchronization

#### 🧠 What I Learned
- How to design database schema using DDL
- Performing various data operations with DML
- Implementing security controls with DCL
- Managing transactions for data consistency

#### 🏆 Achievement
- Built comprehensive SQL training database with Departments and Employees tables
- Implemented all four SQL language categories

#### 📌 Conclusion
SQL fundamentals are essential for database management. Mastered DDL for schema design, DML for data operations, DCL for security, and TCL for transaction control. These core concepts form the foundation for advanced database development.
